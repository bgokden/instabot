# instabot

If you will use a newer version run:
```
./setup.sh
```

To build docker image run:
```
./build.sh
```

Edit env.example.sh with your data

To Run with your env.example.sh configuration:
```
./run.sh example
```

https://github.com/instabot-py/instabot.py
